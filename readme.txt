== Installation ==

Como instalar o WP Plugin Example

= Manual install =
1. Faça o download do WP Plugin Example.
1. Faça o upload do 'WP Plugin Example' para o seu diretório '/wp-content/plugins/'.
1. Ative o WP Plugin Example na sua página Plugins.

= Via WordPress =
Não disponível no momento.
